Detailed Description

This header file contains programming interface for defining regular ocean waves.

This header file provides a wrapper for Boost.Units library for convenience. Coding using Boost.Units can be verbose because of the nested namespaces used. This header file aims at making coding using Boost.Units more concise by providing shorter namespace aliases.

Example of creating an object of type length without using this headder: boost::units::quantity<boost::units::si::length> L = 10 * boost::units::si::meter;

Using this header, the above line can be shortened to: Quantity<Units::length> L = 10 * Units::meter;

This header file also defines some numerical constants which are commonly used.
