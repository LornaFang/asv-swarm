void asv_swarm::Visualisation::Scene::initialise_sea_surface_actor ( Quantity< Units::velocity >  wind_speed,
Quantity< Units::length >  wind_fetch,
Quantity< Units::plane_angle >  wind_direction 
)

Set the sea condition and initialise the sea actor. The method also sets some default values for field size, number of grid points on the field surface, number of wave frequencies considered in the wave spectrum and the number of directions considered in the wave spectrum. The default values can be found in the constructors Hydrodynamics::Sea_surface_dynamics::Sea_surface_dynamics and Hydrodynamics::Wave_spectrum::Wave_spectrum.

Parameters
wind_speedis the wind speed in m/s.
wind_fetchis the length of sea, in m, over which the wind blows.
wind_directionis the direction in which the wind blows measured in radians with respect to North direction and direction of measurement such that East is at PI/2 with North.
